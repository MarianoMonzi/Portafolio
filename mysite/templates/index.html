Mi primer Proyecto con Django y SQLite 3

Django_basic es mi primer proyecto, el cual es una App en la que se pueden agregar y buscar Productos, Empleados y/o Clientes.

La idea principal de la App es poner en practica los conceptos basicos de Django y Sqlite 3. Es por eso que este proyecto no funciona como una Aplicacion CRUD y solo es capaz de añadir datos a la base de datos

Mi primer CRUD con Django y Postgre SQL

Django_CRUD es mi primer CRUD, en el cual se puede agregar tareas, asi como eliminarlas, actualizarlas o marcarlas como completas

El proyecto cuenta con un login, el cual es necesario para poder utilizar la App y asi crear tareas pendientes. Se utilizo Django para crear el Backend, junto a Postgre SQL para la base de datos. La app esta alojada en render.com, ya que es un hosting gratuito.
